
# FactOpedia

This is a Fact sharing web app....


## Run Locally

Clone the project

```bash
  git clone https://github.com/NikhilBhagat01/FactOpedia.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
